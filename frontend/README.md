## eXpServer Tester Utility -- Frontend

## Prerequisites
- `node v18+`


## Installation
- clone the repo 
```bash
git clone https://github.com/eXpServer/expserver-tester.git
```

- cd into frontend directory
```bash
cd frontend
```

- install the dependencies
```bash
npm install
```

- build and execute the program
```bash
npm run build
npm run start
```
